4
Authentication Workflow
User Registration
  1. User provides credentials on signup.php
  2. signupprocessor.php validates input
  3. Password hashing
  4. Store in database
  5. Redirect to login
Login Process
  1. User enters credentials on login.php
  2. loginprocessor.php validates against database
  3. Verifies password hash
  4. Creates secure session
  5. Redirects to dashboard
